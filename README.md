# Tic Tac Toe

This is a tic tac toe hobby project by [Stuxi](https://github.com/iStuxi) and I implemented using LibGDX.

## Building From Source

### Requirements

* Maven
* Java 8+

To build the desktop jar, run this command:
```cmd
> mvn package -P desktop
```
